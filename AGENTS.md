# Generated public profile

## Authority

This repository exposes the designed public experience generated by `greygoody/greygoody-private`.

It is not an authoring or programming surface.

## Agent behavior

When asked to change profile wording, projects, evidence, role positioning, layout, typography, colors, spacing, sections, or assets:

1. do not implement the change here;
2. route the task to `greygoody/greygoody-private`;
3. modify only its approved agent-editable content surface;
4. run its renderer and verifier;
5. let its publishing workflow open or update the public projection pull request.

Agents operating in this repository may inspect, review, compare, or report on the generated experience. They must not directly edit generated files.

## Generated files

The public experience consists of:

- `README.md`
- `assets/profile-light.svg`
- `assets/profile-dark.svg`

These files are accepted only from the `projection/quiet-systems` branch through a pull request carrying private-source provenance.

## Locked repository controls

Do not change:

- `AGENTS.md`
- `.github/`
- repository protection or Actions settings

unless the task is an explicit owner-approved control-plane change.

Direct pushes to `main`, manual edits to generated output, badges, counters, animations, additional profile files, and ad hoc visual experiments are outside this repository's contract.
